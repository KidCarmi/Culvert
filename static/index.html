🔑 Install Root CA — Instructions
Windows:
  1. Double-click proxyshield-ca.pem → Install Certificate
  2. Select "Local Machine" → Place in "Trusted Root CAs"

macOS:
  1. Double-click proxyshield-ca.pem → Keychain Access
  2. Find "ProxyShield Root CA" → Get Info → Trust → Always Trust

Linux (Ubuntu/Debian):
  sudo cp proxyshield-ca.pem /usr/local/share/ca-certificates/proxyshield.crt
  sudo update-ca-certificates

Firefox (all platforms):
  Settings → Privacy → View Certificates → Authorities → Import
